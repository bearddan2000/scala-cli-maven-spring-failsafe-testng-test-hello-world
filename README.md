# scala-cli-maven-spring-failsafe-testng-test-hello-world

## Description
A POC for spring app using testng
framework with failsafe plugin.

## Tech stack
- scala
- maven
  - spring
  - testng
  - failsafe

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
